What are Class Sections?
Class Sections are subdivisions of a main class. For example, a "JSS 1" class might be divided into sections like "JSS 1A", "JSS 1B", and "JSS 1C".

Tip: Create sections before you create your main classes. This will allow you to assign these sections to a class later.
